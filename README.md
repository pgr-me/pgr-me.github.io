# pgr-me.github.io

Personal website and writing archive for **Peter Rasmussen**.

This repository powers my GitHub Pages site at:

**https://pgr-me.github.io**

The site includes a short personal/professional profile, selected technical writing, and archived data science / machine learning projects.

## About the site

This is a lightweight static site built with [Jekyll](https://jekyllrb.com/) and hosted with GitHub Pages.

Most of the older posts in this repository are project writeups from earlier data science work. The site is maintained as a personal homepage and archive rather than as an actively developed application.
